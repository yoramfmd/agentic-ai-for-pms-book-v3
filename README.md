# Agentic AI for Busy Product Managers

**v3.0 · Designing the Agent and the Supervisor**

By Yoram Friedman, MD

A practitioner's guide for product managers building AI products in regulated, high-consequence enterprise domains. Eleven chapters and two appendices on what changes when the software starts acting: how to specify it, how to supervise it, how to know when it is wrong, and what to do before the day it does something nobody on your team knew it could do.

---

## Read the book

The book is hosted at **[agenticaiproductmanagement.com](https://agenticaiproductmanagement.com)**.

Each chapter is a standalone HTML page that shares a single stylesheet (`styles.css`) and a sidebar table of contents. No JavaScript required to read; works offline; prints cleanly.

## Status

This is the v3.0 build. v2.3 (the prior revision) is preserved separately as a Word document, PDF, and Markdown manuscript and is not in this repo.

v3.0 is enriched with content from articles published between January and May 2026, integrates the Supervision Paradox framework, expands the security treatment, deepens the observability vocabulary, and is written for three running case-study threads: software engineering (closest to every PM's daily work), healthcare and clinical AI (sharpest evidence, highest stakes), and SAP enterprise (operational vocabulary).

A complete change log between v2.3 and v3.0 will be published when v3.0 reaches first-draft status.
